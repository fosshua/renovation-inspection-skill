# Evidence Capture Protocol

Use this protocol when image/video evidence is too local, ambiguous, or measurement-dependent. The goal is to reduce visual misclassification and avoid overclaiming.

## Minimum Evidence Pack

For any suspected defect, ask for these four views when missing:

1. **Wide context**: 1-2 meters away, showing room, wall/floor/window/door, and nearby reference points.
2. **Marked location**: same view with tape, finger, pen, or circle marking the exact issue.
3. **Close-up with scale**: ruler, level, feeler gauge, or known object placed on the same plane as the defect.
4. **Side or sweep angle**: side photo or 10-20 second slow video showing depth, plane change, and adjacent surfaces.

If hidden work is involved, add records:
- pressure test, drainage test, water-retention/shower test, conduit threading, concealed-work photos, material labels, or project drawings.

## Measurement Rules

- Treat ruler readings as approximate unless the ruler is on the same plane, perpendicular to the measured edge, and both start/end points are visible.
- State estimated values with tolerance language, for example "约 45mm，受透视影响可能有几毫米误差".
- Do not compare to a numeric tolerance unless the applicable standard, design node, contract, sample board, manufacturer requirement, or enterprise standard provides that tolerance.
- If the defect is workmanship/appearance-based, compare with sample board, agreed drawing, nearby repeated locations, and visible consistency.

## Project Agreement Pack

When national standards do not provide a direct numeric limit, request:

- design node or construction drawing
- signed sample-board photo
- contract or quotation workmanship description
- enterprise standard or project acceptance checklist
- manufacturer installation instructions
- photos of other similar locations for consistency comparison

## Ambiguity Guardrails

- If a close-up could be multiple components, identify candidates and ask for context before final classification.
- User text naming the component or trade should strongly influence stage classification when compatible with the image.
- For terms such as 海棠角, 阳角, 倒角, 碰角, 瓷砖, prioritize tile outside-corner analysis unless the wider image clearly shows a different component.
- Do not let nearby windows, frames, trims, or dark vertical lines override user-labeled tile evidence.

## Stage-Specific Capture Prompts

- **门窗收口**: overall window/door photo, ruler at gap, inner and outer sealant close-ups, side angle, rain/water-spray observation if exposed.
- **贴砖/海棠角**: wide wall/floor photo, marked corner, perpendicular ruler close-up, side angle showing both tile faces, nearby similar corners, sample-board/design detail.
- **水电粗装**: full route photos, crossing close-ups, pressure gauge record, drainage test video, conduit pull-through evidence, before-closure archive.
- **防水**: before/after coating wide photos, pipe-root/corner/threshold close-ups, upturn-height ruler photo, curing/protection state, water-retention or shower-test record.
- **竣工验收**: room-by-room walkthrough, water/drainage function videos, switch/socket tests, door/window operation videos, punch-list closure evidence.
